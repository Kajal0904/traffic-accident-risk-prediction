# Traffic Accident Risk Prediction

This project predicts traffic accident risk using Machine Learning, Fuzzy Logic, and Bayesian Belief Networks (BBN).

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Fuzzy Logic

## Features
- Traffic risk prediction
- Data visualization
- Fuzzy inference system
- Machine Learning models
- Bayesian reasoning
